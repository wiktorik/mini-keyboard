 1. Set arduino IDE
1. Open Arduino IDE.
2. Go to Settings next to the Additional Boards Manager URLs and add:
   https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
3. Install ESP32 by Espressif in Boards Manager.
4. Select board: ESP32C3 Dev Module.

2. Library Installation
Search and install these in the Library Manager:
- Adafruit SSD1306
- Adafruit SSD1306
- Adafruit GFX Library

3. Flashing
Connect your ESP32 Super Mini via USB-C and click upload.
